# Lottery
**A lottery application which is small but strong.**

## Requirement
**A Windows equipment with `.NET Framework 4.8` or higher.**

However, sometimes you can still use it without the required environment due to some `special things`.

## Build & Run
**We advise you to use `Visual Studio` to build the project.**

If you only want to use it, please go to the [Release Page](https://github.com/Unqualified-Developers/Lottery/releases/tag/v1.0.0.2) or use the direct links below. The latest release is `v1.0.0.2`.

[Executable Program (AMD64)](https://github.com/Unqualified-Developers/Lottery/releases/download/v1.0.0.2/Lottery-amd64.exe)\
[Executable Program (Any CPU)](https://github.com/Unqualified-Developers/Lottery/releases/download/v1.0.0.2/Lottery.exe)
